Install
------------------------------------------------------------------------------

``whoosh_examples`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install whoosh_examples

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade whoosh_examples
